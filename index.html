ISS-Tracker by Team Bhimwal

Welcome to International Space Station (ISS) tracker. This project tracks ISS in real-time along with displaying other details such as time, latitude, longitude etc. We built this web app using JavaScript, Leaflet JavaScript library and WTIA REST API.

We use Leaflet JS map inside of which we draw a marker. ISS image is our marker. Then every 2 seconds, we perform a GET request to the api.wheretheiss.at, which in turn returns all the information like current latitude, longitude, timezone, speed, altitude, and visibility of ISS. Then we update the position of the marker (ISS image) on the map and all the details according to the information we get.

Check out NASA's website to find out more about ISS!
